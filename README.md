# SecureIT Consulting

A hands-on cybersecurity and IT infrastructure portfolio project simulating the deployment of a secure network for a fictional healthcare organization.

## Project Overview

This project demonstrates practical experience with:

- Client discovery
- Asset inventory
- Network design
- Network segmentation
- IP addressing
- Active Directory deployment
- Organizational Units (OUs)
- User and Group management
- Group Policy Objects (GPO)
- NTFS permissions
- Shared folders and access control

## Environment

- Windows Server 2022
- Windows 10 Client
- Active Directory Domain Services (AD DS)
- Group Policy Management
- VirtualBox
- Draw.io

## Active Directory Highlights

The Active Directory environment includes:

- Healthcare organizational structure
- Departmental Organizational Units
- Department security groups
- Employee accounts
- Password policies
- Folder permissions using NTFS
- Shared network resources
- Access testing from a Windows client
