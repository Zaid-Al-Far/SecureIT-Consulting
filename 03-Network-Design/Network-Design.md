# Network Design

## Objective

Design a secure network architecture for HealthyCare Medical Clinic.

## Network Components

- Internet
- ISP Modem
- pfSense Firewall
- Managed Switch
- Domain Controller
- File Server
- 30 Windows Workstations
- Wireless Access Point
- Network Printer

## Design Overview

The clinic network is designed using a layered architecture. Internet traffic enters through the ISP modem and passes through the pfSense firewall before reaching the managed switch. The switch connects all internal devices, including servers, workstations, printers, and the wireless access point.

This design provides centralized management, improved security, and scalability for future growth.
