# Active Directory Lab

## Objective
Build an Active Directory environment for a healthcare clinic using Windows Server 2022.

## Tasks Completed
- Installed Active Directory Domain Services (AD DS)
- Created the lab.local domain
- Joined a Windows 10 client to the domain
- Created Organizational Units (OUs)
- Created security groups
- Created user accounts
- Applied Group Policy Objects (GPOs)
- Configured shared folders
- Applied NTFS permissions
- Tested user access based on group membership

## Environment
- Windows Server 2022
- Windows 10
- VirtualBox
- Active Directory Domain Services
- Group Policy Management
